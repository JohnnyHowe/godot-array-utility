# Overview
Provides a bunch of useful things to do with GDScript `Array` objects.

# Install
Just download and drop in to your project.

Or as a git submodule
```
git submodule add https://github.com/JohnnyHowe/godot-array-utility addons/array_utility
```

# Dependencies
## gdscript-script-test-runner
This is purely for testing. If you don't care, just delete the "tests" folder.

Installing the test runner is the same as this package.

Download it and drop it in. (download here: https://github.com/JohnnyHowe/gdscript-script-test-runner)

or via git submodule
```
git submodule add https://github.com/JohnnyHowe/gdscript-script-test-runner addons/gdscript_script_test_runner
```
